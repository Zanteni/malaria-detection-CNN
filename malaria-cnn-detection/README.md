#Malaria CNN Detection
This project implements a Convolutional Neural Network (CNN) using PyTorch to detect malaria-infected cells from microscope images.

##Model Performance
Training Accuracy: 96.01%

Validation Accuracy: 95.81%

Test Accuracy: 95.26%

Training Loss: 0.1134

Validation Loss: 0.1359

Test Loss: 0.1377

Overfitting Gap: max ~0.20%
