# Malaria CNN Detection

This project implements a Convolutional Neural Network (CNN) using **PyTorch** to detect malaria-infected cells from microscope images.

## Model Performance

- **Training Accuracy:** 95.82%  
- **Validation Accuracy:** 95.60%  
- **Test Accuracy:** 95.24%  

- **Training Loss:** 0.1268  
- **Validation Loss:** 0.1619  
- **Test Loss:** 0.1418  

- **Overfitting Gap:** max ~0.59%
